# 问题6.1

假设 S 和 T 都是向量空间 V 的子空间

  1. 定义：S+T 是包含所有 s+t 组合的向量集合，其中 s 在子空间 S 中，t 在子空间 T 中。求证：S+T 是一个向量空间（满足对于线性组合封闭）

  2. 如果 S 和 T 都是 Rm 向量空间中的一条直线，那么 S+T 和 S∪T 的区别是什么。（即解释 S+T 是由 S∪T 张成得到的）

  1. 解释

假设 s′ 是子空间 S 的向量，t′ 是子空间 T 的向量，则有

s+t+s′+t′=(s+s′)+(t+t′)​

根据向量空间对于向量的加法的封闭性，可得 s+s′∈S 和 t+t′∈T

所以 (s+s′)+(t+t′)∈S+T 即 S+T 对向量的加法封闭。

类似地，根据向量空间对于向量的数乘的封闭性，可得 cs∈S 和 ct∈T，其中 c 是常量，所以 c(s+t)=cs+ct∈S+T

所以 S+T 对于线性组合封闭。

  1. 解释 当 S 和 T 是两条不同的直线时，则根据 S+T 的定义，该向量集合的几何形式是一个平面；而根据 S∪T 的定义，该向量集合的几何形式仅是两条直线。而 S∪T 张成的空间是两个向量 s 和 t 的所有可能的线性组合，即 s+t，其中 s 是在直线 S 上，t 是在 T 上。

所以 S+T 是由 S∪T 张成得到的。

# 问题6.2

已知目标平面 x−3y−z=12 与平面 x−3y−z=0 平行，其中向量 (12,0,0) 在目标平面上。

则在目标平面上的点具有以下模式，请填充完整：

⎣⎡​xyz​⎦⎤​=⎣⎡​?00​⎦⎤​+y⎣⎡​?10​⎦⎤​+z⎣⎡​?01​⎦⎤​​

解法一：

两平面平行，且平面 x−3y−z=0 过原点 (0,0,0)，而点 (12,0,0) 在目标平面，因此从几何角度考虑，目标平面上的点可以将平面 x−3y−z=0 上的点沿 x 轴移动 12 个单位得到相应的点。

以矩阵的角度考虑，其中 ⎣⎡​xyz​⎦⎤​ 是目标平面上的点，⎣⎡​x′y′z′​⎦⎤​ 是平面 x−3y−z=0 上的点:

⎣⎡​xyz​⎦⎤​=⎣⎡​1200​⎦⎤​+⎣⎡​x′y′z′​⎦⎤​​

将平面（方程组） x−3y−z=0 写成矩阵形式 Ax=b

[1​−3​−1​]⎣⎡​xyz​⎦⎤​=0​

这些向量都可以使方程组 Ax=0 成立，则平面上的点构成零空间 N(A)，根据题目的提示可以写出其中两个向量 ⎣⎡​310​⎦⎤​ 和 ⎣⎡​101​⎦⎤​

可以将这两个向量作为基向量 basis 写出零空间向量的一般形式

⎣⎡​x′y′z′​⎦⎤​=m⎣⎡​310​⎦⎤​+n⎣⎡​101​⎦⎤​​

而题目要求的形式则是将解进行「分解」，消去 x

即

⎣⎡​x′y′z′​⎦⎤​=y′⎣⎡​310​⎦⎤​+z′⎣⎡​101​⎦⎤​​

因此可得

⎣⎡​xyz​⎦⎤​=⎣⎡​1200​⎦⎤​+y⎣⎡​310​⎦⎤​+z⎣⎡​101​⎦⎤​​

解法二： 直接使用换元，并将写成矩阵形式

⎣⎡​xyz​⎦⎤​=(⎣⎡​12+3y+zyz​⎦⎤​)=⎣⎡​12+0+00+y+00+0+z​⎦⎤​=⎣⎡​1200​⎦⎤​+y⎣⎡​010​⎦⎤​+z⎣⎡​001​⎦⎤​​

# 问题6.3

若向量空间 C 由向量空间 A 和 B 构成，且满足 C=[AB​]，求它们的零空间 N(C) 和 N(A) 与 N(B) 的关系。

已知 C=[AB​] 则对于方程组 Cx=0 的所有解构成的零空间 N(C)，需要满足 Cx=[AxBx​]=0，即这些解 x 需要同时满足 Ax=0 和 Bx=0，所以 N(C)=N(A)∩N(B)
