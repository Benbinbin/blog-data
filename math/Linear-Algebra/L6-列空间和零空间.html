# L6-列空间和零空间

# 列空间 C(A)

对于矩阵 A

A=⎣⎡​1234​1111​2345​⎦⎤​​

其列空间 C(A) 是由矩阵 A 的 3 个列向量的所有线性组合构成的向量集合，记作 N(A)。它是 R4 的子空间，而且由于 A 是 4×3 矩阵（只有 3 个列向量），因此该列空间是 R4 的真子空间。

💡 列空间是为了研究方程组（矩阵形式） Ax=b 是否有解 这类问题构建出来的。

Ax=⎣⎡​1234​1111​2345​⎦⎤​⎣⎡​xyz​⎦⎤​=⎣⎡​b1​b2​b3​b4​​⎦⎤​=b​

对于示例的方程组，当解存在时（不同情况），求向量 b 的特点：

  1. 当 b=⎣⎡​0000​⎦⎤​ 时有解，解为 x=⎣⎡​000​⎦⎤​

  2. 换另一种角度，可以先给出 x 值，通过 Ax 计算得到 b，则当 b 为该值时，必然有对应的 x 值，即存在解。

    因此当 x=⎣⎡​111​⎦⎤​ 时，得到 b=⎣⎡​46810​⎦⎤​

    还有很多不同的 x 值，对应可以得到相应的 b 值。

  3. 从矩阵与向量相乘的角度考虑，结果向量 b 就是系数矩阵 A 各列基于 x 作为系数进行线性组合得到的，因此只要 b 在列空间 C(A) 中，则方程组 Ax=b 就有解。即当向量 b 在列空间 C(A) 中，必然存在系数使得矩阵各列进行线性组合得到 b，而该线性组合的系数就是解 x 的值。

💡 对于方程组 Ax=b，观察系数矩阵 A，可知 col3=col1+col2

⎣⎡​2345​⎦⎤​​=⎣⎡​1234​⎦⎤​+⎣⎡​1111​⎦⎤​​

因此矩阵 A 各列并不独立（它们线性相关），所以列空间 C(A) 实际是由其中两个向量张成得到的（另一个向量并无「贡献」）是 R4 中的二维子空间。

# 零空间 N(A)

零空间 Null Space 是指使方程组 Ax=0 成立的所有（列向量） x 构成的向量集合，记作 N(A)。注意方程组的特殊形式，相当于 Ax=b 中使得 b=0。

Ax=⎣⎡​1234​1111​2345​⎦⎤​⎣⎡​xyz​⎦⎤​​=0​

可以尝试写出一些使得等式成立的解（其中零向量必然使等式成立）

x=⎣⎡​000​⎦⎤​,x=⎣⎡​−1−11​⎦⎤​,x=⎣⎡​11−1​⎦⎤​​

观察系数矩阵 A，可知 col3=col1+col2，因此可以写出解的一般形式为 c⎣⎡​11−1​⎦⎤​，其中 c 是系数。

所有使等式 Ax=0 成立的解 x 构成向量集合，其几何形式是 R3 中过原点的一条直线，向量 ⎣⎡​11−1​⎦⎤​ 在直线上。

Ax=0 的所有解 x 的集合，总是可以构成向量空间（子空间），即零空间 N(A) 是一个向量空间

💡 假设存在两个解 v 和 w 使得等式 Ax=0 成立，则由 Av+Aw=A(v+w)=0 成立，所以 v+w 也是等式 Ax=0 的解，即 v+w 仍在零空间中，所以零空间对于向量相加运算是封闭的；类似地，零空间对于向量的数乘运算也是封闭的。所以 Ax=0 的所有解 x 构成的向量集合，零空间，是一个向量空间。

⚠️ 如果对于以上例子 Ax=b 等式中，当 b=0 时，由等式的所有解 x 构成的向量集合，并不能构成向量空间，由于零向量并不包含在该集合中，如果 x 维度是 R3，则所有解构成的集合的几何形状是一条不过原点的直线。

# C(A) vs N(A)

对于矩阵 A

A=⎣⎡​1234​1111​2345​⎦⎤​​
